# 🚀 K8s (Kubernetes Learning Repository)

This repository contains foundational concepts and practical commands to help understand and work with Kubernetes effectively.

---

## 📂 Project Structure

```
k8s/
│
├── basics/
│   ├── architecture.md     # Overview of Kubernetes architecture
│   └── common_commands.md         # Common kubectl commands cheat sheet
│
└── README.md
```

---

## 📘 Contents

### 🔹 Basics

#### 1. Architecture
- High-level overview of Kubernetes components
- Control Plane (API Server, Scheduler, Controller Manager)
- Worker Nodes (Kubelet, Kube Proxy, Pods)
- Cluster communication flow

#### 2. Common Commands
- Essential `kubectl` commands
- Cluster management
- Resource inspection
- Debugging and troubleshooting
- Scaling and deployments

---

## ⚙️ Prerequisites

Before using this repo, make sure you have:

- Kubernetes cluster (Minikube / Kind / Cloud)
- `kubectl` installed and configured
- Basic understanding of containers (Docker helps)

---

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/k8s.git
cd k8s
```

Explore the basics:

```bash
cd basics
```
